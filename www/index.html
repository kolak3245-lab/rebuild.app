Cara Kerja (Real)

Alur build yang sebenarnya terjadi saat kamu klik "Mulai Build Nyata":

1Browser kirim request ke GitHub API untuk buat repo baru
2File HTML kamu + config.xml + workflow YAML di-push ke repo
3GitHub Actions otomatis berjalan di server GitHub (bukan HP kamu)
4Cordova + Android SDK mengkompilasi APK debug (~3-5 menit)
5Download APK dari halaman Actions → Artifacts di repo GitHub
Token GitHub hanya tersimpan di memori browser. Tidak pernah dikirim ke server selain GitHub.
